CPU Scheduling Simulator

An interactive visualization tool to simulate and compare different Operating System CPU scheduling algorithms.

Fully Implemented

First Come First Serve (FCFS)

Non-preemptive algorithm that schedules processes strictly based on their arrival time. Simpler, but prone to the Convoy Effect.

Launch FCFS
Fully Implemented

Shortest Job First (SJF)

Non-preemptive algorithm that selects the process with the shortest burst time. Minimizes average waiting time.

Launch SJF
Fully Implemented

Shortest Remaining Time (SRT)

Preemptive version of Shortest Job First scheduling. Dynamically preempts currently running process if a shorter job arrives.

Launch SRT
Fully Implemented

Round Robin (RR)

Preemptive algorithm that allocates a fixed time slice (quantum) to each process in cyclic order. Ensures fairness.

Launch RR
Fully Implemented

Multilevel Feedback Queue (MLFQ)

A dynamic scheduler using multiple priority queues with differing time quantum values, aging, and promotion/demotion logic.

Launch MLFQ
